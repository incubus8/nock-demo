## Nock Demo

This is the demo for how to use "Nock" to help integration testing.

### How to run

1. Install dependencies, type `npm i`
2. Run `npm start`.

You will see the following results:

```shell
➜ npm start

> nested-nock-demo@1.0.0 start /Users/angel/Work/nested-nock
> node index.js

{ status: 'up' }
{ meat: [ 'beef', 'chicken' ] }
```
